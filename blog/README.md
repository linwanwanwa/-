# Linwan's Blog

个人博客网站，包含项目展示、博客文章和联系方式。

## 📁 项目结构

```
blog/
├── pages/                      # 页面文件
│   ├── index.html             # 首页（个人主页）
│   ├── 404.html               # 404错误页面
│   ├── 原谅我.html              # 道歉页面
│   └── 长沙两天行程规划（最终版）.html  # 长沙旅游行程规划
├── assets/                     # 资源文件
│   ├── css/                   # 样式文件
│   │   └── styles.css         # 主样式表
│   └── images/                # 图片资源
│       └── 头像.png            # 个人头像
└── apps/                       # 应用程序
    └── 鼠标连点器/             # 鼠标连点器工具
        ├── 说明.txt
        └── 鼠标连点器.exe
```

## 🚀 快速开始

### 本地开发

在项目根目录下运行：

```bash
python -m http.server 8000
```

然后在浏览器中访问：http://localhost:8000/pages/index.html

### 文件说明

#### pages/ - 页面目录
- **index.html**: 网站首页，包含个人介绍、项目展示、博客列表和联系方式
- **404.html**: 404错误页面，用于处理无效链接
- **原谅我.html**: 特殊的道歉页面，包含爱心飘落动画效果
- **长沙两天行程规划（最终版）.html**: 长沙两日游详细行程规划，包含网红打卡点和实用贴士

#### assets/ - 资源目录
- **css/styles.css**: 全局样式表，支持日夜主题切换，包含响应式设计
- **images/头像.png**: 个人头像图片

#### apps/ - 应用目录
- **鼠标连点器/**: 实用工具程序

## ✨ 特性

- 🌓 **日夜主题切换**: 支持浅色/深色主题，自动检测系统偏好
- 📱 **响应式设计**: 完美适配手机、平板和桌面设备
- 🎨 **精美动画**: 滚动动画、爱心飘落等交互效果
- ⚡ **性能优化**: 资源预加载、异步加载、Intersection Observer API
- 🎯 **轻量级**: 无需构建工具，纯HTML/CSS/JavaScript
